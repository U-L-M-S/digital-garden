> Zugriffsrechte.
> Es bestimmt, wie weit ein [[02 - RESOURCES/Notes/Attribut\|Attribut]] oder eine [[02 - RESOURCES/Notes/Programmierung Methode\|Programmierung Methode]] zugänglich ist.

Zugriffsmodifikatoren können `public`, `private`, `protected` sein. 
Es hängt nur davon ab, von wo man auf das [[02 - RESOURCES/Notes/Attribut\|Attribut]] / die [[02 - RESOURCES/Notes/Programmierung Methode\|Programmierung Methode]] zugreifen möchte.

| Zugriffsmodifikator | Zugriff                                                                          |
| :-----------------: | :------------------------------------------------------------------------------- |
|       public        | Überall zugreifbar                                                               |
|       private       | Nur in derselben [[02 - RESOURCES/Notes/Klasse\|Klasse]], in der es erstellt wurde, zugreifbar                |
|      protected      | Nur in derselben [[02 - RESOURCES/Notes/Klasse\|Klasse]], in der es erstellt wurde, und in den Subklassen zugreifbar |
