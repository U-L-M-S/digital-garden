>Mit diesem Befehl kannst du Ordner / Files auswählen, die danach im Save Point mit `commit` gespeichert werden.
>>Mit `git add file1 file2` werden mehrere Dateien ausgewählt, und mit `git add *` ODER `git add .` werden ==alle== Dateien und Verzeichnisse ausgewählt.
```bash
git add <file>
```
