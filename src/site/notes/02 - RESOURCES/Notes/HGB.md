>Das HGB (Handelsgesetzbuch) ist das Gesetz für Geschäfte zwischen Unternehmen (Kaufleuten).
>>Es ergänzt das [[02 - RESOURCES/Notes/BGB\|BGB]] mit speziellen Regeln für gewerbliche Tätigkeiten und Handel.

>[!summary] Was macht das HGB?
>- Regelt **Geschäfte zwischen Firmen** (B2B = Business-to-Business)
>- Definiert wer ein **Kaufmann** ist
>- Strengere Regeln als im [[02 - RESOURCES/Notes/BGB\|BGB]]

```
BGB vs HGB

┌─────────────────┐         ┌─────────────────┐
│      BGB        │         │      HGB        │
│  Privatrecht    │         │  Handelsrecht   │
└─────────────────┘         └─────────────────┘
        │                           │
        │                           │
   Für ALLE                   Nur für KAUFLEUTE
        │                           │
        ▼                           ▼
┌─────────────────┐         ┌─────────────────┐
│ • Privatpersonen│         │ • GmbH, AG      │
│ • Unternehmen   │         │ • eingetragene  │
│ • Vereine       │         │   Kaufleute     │
└─────────────────┘         └─────────────────┘

           HGB ergänzt/erweitert BGB!
```

>[!important] HGB-Bereiche für IT-Unternehmen
>**Wichtige Pflichten:**
>- **Kaufmannseigenschaft:** Wer ist Kaufmann?
>- **[[02 - RESOURCES/Notes/Handelsregister\|Handelsregister]]:** Eintragungspflicht
>- **Buchführungspflicht:** Genaue Aufzeichnungen führen
>- **Handelsbräuche:** Besondere Geschäftsregeln

```
KAUFMANN-ARTEN

┌───────────────────────────────────────┐
│  Ist-Kaufmann (§ 1 HGB)               │
│  → Gewerbebetrieb braucht kaufmännische│
│     Einrichtung                        │
│  Beispiel: IT-GmbH, Software-AG       │
└───────────────────────────────────────┘
               ↓
┌───────────────────────────────────────┐
│  Kann-Kaufmann (§ 2 HGB)              │
│  → Kleingewerbe, kann sich eintragen  │
│  Beispiel: Kleiner Freelancer         │
└───────────────────────────────────────┘
               ↓
┌───────────────────────────────────────┐
│  Form-Kaufmann (§ 6 HGB)              │
│  → GmbH, AG sind IMMER Kaufleute      │
│  Beispiel: Software GmbH              │
└───────────────────────────────────────┘
```

>[!example] IT-Unternehmen im HGB
>**Typische Situation:**
>```
>IT-Firma GmbH (Kaufmann)
>        ↓
> Verkauft Software an
>        ↓
>Kunden-Firma AG (auch Kaufmann)
>
>→ HGB gilt! (B2B-Geschäft)
>→ Strengere Regeln
>→ Schnellere Fristen
>→ Buchführungspflicht
>```

>[!warning] Unterschiede BGB ↔ HGB
>**Was ändert sich bei Kaufleuten:**
>
>| Regel | [[02 - RESOURCES/Notes/BGB\|BGB]] | HGB |
>|-------|-----|-----|
>| **Vertragsbestätigung** | Nicht nötig | Schweigen = Zustimmung (§ 362 HGB) |
>| **Mängelrüge** | Angemessene Zeit | Sofort prüfen! (§ 377 HGB) |
>| **Zinsen** | 5% über Basiszins | 9% über Basiszins |

>[!tip] Praxis-Tipp für die Prüfung
>**Faustregel:**
>- Privat an Privat = [[02 - RESOURCES/Notes/BGB\|BGB]]
>- Firma an Privat = [[02 - RESOURCES/Notes/BGB\|BGB]]
>- **Firma an Firma = HGB** ⭐
