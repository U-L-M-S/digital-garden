>Ein Angebot ist die Menge an Produkten oder Dienstleistungen, die Verkäufer zu bestimmten Preisen verkaufen möchten.
>>Die Verkäuferseite trifft auf die [[02 - RESOURCES/Notes/Nachfrage\|Nachfrage]] (Käuferseite) und zusammen bildet sich der Marktpreis.

>[!summary] Kernpunkte
>- **Anbieter:** Verkäufer, die Waren/Dienstleistungen verkaufen wollen
>- **Preis:** Je höher der Preis, desto mehr wollen Verkäufer anbieten
>- **Markt:** Zusammenspiel von Angebot und [[02 - RESOURCES/Notes/Nachfrage\|Nachfrage]]

>[!example] Beispiel aus der IT
>**Software-Entwicklung:**
>- Angebot: IT-Unternehmen bietet Programmierung einer App an
>- Preis: 50.000€ für Entwicklung
>- Leistung: Fertige App mit allen vereinbarten Funktionen

```
ANGEBOTSKURVE (Wirtschaft)

Preis (€)
  ↑
  │         ╱
  │       ╱
  │     ╱    (Angebot steigt
  │   ╱      bei höheren Preisen)
  │ ╱
  └────────────→ Menge

  Mehr Preis = Mehr Angebot
```

>[!important] Rechtliche Bedeutung (IT-Verträge)
>Im IT-Bereich ist ein **Angebot** eine verbindliche Willenserklärung:
>- Muss konkret sein (Leistung, Preis, Zeit)
>- Bindet den Anbieter rechtlich (§ 145 [[02 - RESOURCES/Notes/BGB\|BGB]])
>- Nach Annahme entsteht ein Vertrag
