# Jugendliche
>Lautet [[02 - RESOURCES/Notes/jArbSchG\|jArbSchG]]. Jeder der unter 18 ist soll 8 Stunden wöchentlich arbeiten.
>Mit Ausnahmen von 8,5 Stunden und Überstunden sollen innerhalb von 3 Wochen ausgeglichen werden.
