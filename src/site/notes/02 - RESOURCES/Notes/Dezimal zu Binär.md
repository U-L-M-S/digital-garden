>Es gibt mehrere Möglichkeiten, aber oft wird das hier verwendet:

96 zu Binär

| Zahl  | :2  | Ergebnis | Rest |
| :---: | :-: | :------: | :--: |
|  96   | :2  |    48    |  0   |
|  48   | :2  |    24    |  0   |
|  24   | :2  |    12    |  0   |
|  12   | :2  |    6     |  0   |
|   6   | :2  |    3     |  0   |
|   3   | :2  |    1     |  1   |
|   1   | :2  |    1     |  1   |
| **0** |     |          |      |

Jetzt müssen wir den Rest **rückwärts** zählen.
D.h: **96** in Dezimal ist **1100000**
