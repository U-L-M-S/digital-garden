>Menge an Gütern, die Käufer zu verschiedenen Preisen kaufen möchten.
>>Käuferseite - bestimmt zusammen mit [[02 - RESOURCES/Notes/Angebot\|Angebot]] den Marktpreis.

>[!summary] Zusammenfassung
>**Nachfrage** beschreibt die Bereitschaft und Fähigkeit von Käufern, ein Gut zu einem bestimmten Preis zu erwerben. Sie ist das **Gegenstück zum [[02 - RESOURCES/Notes/Angebot\|Angebot]]** und bildet gemeinsam den **Marktmechanismus**. Die Nachfrage folgt dem **Gesetz der Nachfrage**: Je höher der Preis, desto geringer die nachgefragte Menge (und umgekehrt).

## Nachfrage vs. Angebot: Vergleichstabelle

| **Aspekt**          | **Nachfrage**                  | **Angebot**                    |
| ------------------- | ------------------------------ | ------------------------------ |
| **Perspektive**     | Käufer/Konsument               | Verkäufer/Produzent            |
| **Frage**           | "Was will ich kaufen?"         | "Was kann ich verkaufen?"      |
| **Preisreaktion**   | Preis ↑ → Nachfrage ↓          | Preis ↑ → Angebot ↑            |
| **Kurvenverlauf**   | Fallend (von links oben)       | Steigend (von links unten)     |
| **Wirtschaftsziel** | Nutzenmaximierung              | Gewinnmaximierung              |
| **Einflussfaktoren**| Einkommen, Präferenzen, Trends | Produktionskosten, Technologie |

## Marktmechanismus: Wie Angebot und Nachfrage interagieren

```
SITUATION 1: ANGEBOTSÜBERSCHUSS
─────────────────────────────────
         Angebot > Nachfrage
              │
              ↓
    Preis SINKT (Rabatte!)
              │
              ↓
      Nachfrage steigt
              │
              ↓
     ← Marktgleichgewicht →


SITUATION 2: NACHFRAGEÜBERSCHUSS
─────────────────────────────────
         Nachfrage > Angebot
              │
              ↓
     Preis STEIGT (Knappheit!)
              │
              ↓
       Angebot steigt
              │
              ↓
     ← Marktgleichgewicht →
```

>[!important] Wichtig für AP2
>- **Nachfragegesetz**: Je höher der Preis, desto geringer die Nachfrage
>- **Marktgleichgewicht**: Schnittpunkt von Angebots- und Nachfragekurve
>- **Preiselastizität**: Wie stark reagiert die Nachfrage auf Preisänderungen?
>  - **Elastisch**: Kleine Preisänderung → große Mengenänderung (z.B. Luxusgüter)
>  - **Unelastisch**: Große Preisänderung → kleine Mengenänderung (z.B. Grundnahrungsmittel)

>[!example] Beispiel: Smartphone-Markt

**Szenario**: Neues iPhone wird vorgestellt

```
VOR Markteinführung:
────────────────────
Preis: 1200€
Nachfrage: HOCH (Hype!)
Angebot: NIEDRIG (limitiert)
→ Nachfrageüberschuss
→ Preis könnte steigen


6 Monate NACH Markteinführung:
───────────────────────────────
Preis: 900€ (Rabatte)
Nachfrage: MITTEL (Hype vorbei)
Angebot: HOCH (mehr Produktion)
→ Angebotsüberschuss
→ Preis sinkt
```

**Nachfragekurve für Smartphone**:

## Einflussfaktoren auf die Nachfrage

```
FAKTOREN, DIE NACHFRAGE ERHÖHEN:
┌──────────────────────────────────┐
│ • Einkommen steigt               │ → Nachfragekurve verschiebt
│ • Positive Trends/Mode           │    sich nach RECHTS
│ • Preis von Substituten steigt   │
│ • Marketing/Werbung              │         Preis
│ • Bevölkerungswachstum           │          │
└──────────────────────────────────┘          │  D₁  D₂
                                              │   \ \
FAKTOREN, DIE NACHFRAGE SENKEN:               │    \ \
┌──────────────────────────────────┐          │     \ \
│ • Einkommen sinkt                │          └──────→ Menge
│ • Negative Publicity             │          Alt → Neu
│ • Preis von Komplementen steigt  │
│ • Präferenzänderung              │
└──────────────────────────────────┘
```

>[!tip] Praxis-Tipps für AP2
>1. **Kurvenverlauf merken**: Nachfragekurve fällt von links oben nach rechts unten (wie ein Abhang)
>2. **Eselsbrücke**: "Teuer = Wenige kaufen, Billig = Viele kaufen"
>3. **Marktgleichgewicht**: Immer am Schnittpunkt der beiden Kurven
>4. **Verschiebung vs. Bewegung**:
>   - **AUF der Kurve**: Preisänderung
>   - **VERSCHIEBUNG der Kurve**: Andere Faktoren (Einkommen, Trends, etc.)
>5. **Typische Prüfungsfrage**: "Was passiert mit dem Preis, wenn die Nachfrage steigt?"
>   → Antwort: Preis steigt (bei gleichbleibendem Angebot)
