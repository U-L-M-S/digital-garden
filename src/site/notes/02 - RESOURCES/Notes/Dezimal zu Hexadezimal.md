>Dezimalzahl durch 16 teilen, Reste notieren, rückwärts schreiben:

Beispiel: **431**

|   |   |   |   |
|---|---|---|---|
|Zahl|:16|Ergebnis|Rest|
|431|:16|26|15 (F)|
|26|:16|1|10 (A)|
|1|:16|0|1|

Also ist **431** in [[02 - RESOURCES/Notes/Hexadezimal\|Hexadezimal]] **1AF**.
