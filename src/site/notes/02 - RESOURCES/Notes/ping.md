> Befehl, der verwendet wird, um zu überprüfen, ob zwei Geräte miteinander verbunden sind.
> Es verwendet das Protokoll [[02 - RESOURCES/Notes/ICMP\|ICMP]].


```sh
ping 192.168.1.69
```
Oder über DDNS
```sh
ping www.google.com
```
