>AGB (Allgemeine Geschäftsbedingungen) sind vorgefertigte [[02 - RESOURCES/Notes/Vertrag\|Vertrag]]sregeln, die ein Unternehmen für alle Kunden verwendet.
>>Sie müssen klar verständlich, fair und rechtlich erlaubt sein (geregelt durch das [[02 - RESOURCES/Notes/BGB\|BGB]] §§ 305-310).

>[!summary] Was sind AGB?
>- **Vorgefertigte Regeln** für viele Verträge
>- **Standard-Bedingungen** die für alle Kunden gelten
>- Müssen **transparent und fair** sein

```
VERTRAGSBILDUNG MIT AGB

Anbieter                    Kunde
    │                          │
    ├─ Angebot mit AGB ───────>│
    │  (Link/Anhang)            │
    │                           │
    │<─── Kunde liest AGB ──────┤ (sollte!)
    │                           │
    │<─── Kunde akzeptiert ─────┤ ☑️
    │                           │
    └───── Vertrag + AGB ───────┘
```

>[!important] AGB-Anforderungen (§§ 305-310 BGB)
>**4 wichtige Regeln:**
>
>1. **Einbeziehung:** Kunde muss AGB kennen können
>   - Vor Vertragsschluss zugänglich machen
>   - Hinweis auf AGB
>
>2. **Transparenzgebot:** Klar und verständlich
>   - Keine komplizierten Juristenformulierungen
>   - Deutsche Sprache
>
>3. **Keine überraschenden Klauseln**
>   - Ungewöhnliche Regeln sind unwirksam
>   - Kunde muss damit rechnen können
>
>4. **Angemessenheit**
>   - Nicht zu einseitig für Anbieter
>   - Kunde darf nicht benachteiligt werden

>[!example] AGB im IT-Bereich
>**Typische Inhalte in Software-AGB:**
>```
>• Nutzungsrechte (Lizenz)
>• Haftungsausschlüsse
>• Gewährleistung
>• Zahlungsbedingungen
>• Kündigungsfristen
>• Datenschutz
>```

```
AGB-KONTROLLE (Gericht prüft)

┌─────────────────────────────────────┐
│  Ist die Klausel klar formuliert?   │
│          (Transparenz)               │
└─────────────────────────────────────┘
               ↓ JA
┌─────────────────────────────────────┐
│  Ist die Klausel überraschend?      │
│       (§ 305c BGB)                  │
└─────────────────────────────────────┘
               ↓ NEIN
┌─────────────────────────────────────┐
│  Benachteiligt sie den Kunden       │
│    unangemessen? (§ 307 BGB)        │
└─────────────────────────────────────┘
               ↓ NEIN
┌─────────────────────────────────────┐
│      ✓ Klausel ist WIRKSAM          │
└─────────────────────────────────────┘
```

>[!warning] Unwirksame AGB-Klauseln
>**Diese Klauseln sind oft unwirksam:**
>- Totaler Haftungsausschluss
>- Kunde trägt alle Risiken
>- Einseitiges Änderungsrecht
>- Überlange Vertragsbindung
>
>**Folge:** Unwirksame Klausel wird gestrichen, Rest bleibt gültig!

>[!tip] Praxis-Tipp für die Prüfung
>**Im Zweifel gilt:** Unklare AGB werden **gegen den Verwender** ausgelegt!
>(§ 305c Abs. 2 [[02 - RESOURCES/Notes/BGB\|BGB]])
